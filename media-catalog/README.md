# Media Catalog

MVP para catálogo bonito de la colección usando como fuente el WebDAV de AltMount.

## Objetivo
- Indexar colección desde WebDAV/estructura montada
- Enriquecer con metadata básica derivada de la ruta
- Exponer una web moderna y bonita
- Usar Telegram como interfaz rápida de búsqueda

## Componentes previstos
- `web/` → frontend React/Vite
- `docs/` → notas de arquitectura y roadmap
