# Roadmap MVP

## Fase 1
- Fuente: árbol WebDAV de AltMount
- Extraer:
  - tipo (película/serie)
  - título
  - año
  - temporada/episodio
  - calidad inferida por ruta/nombre
- Persistencia inicial: JSON mock o SQLite
- Web bonita con:
  - home
  - búsqueda
  - ficha detalle
  - secciones de películas / series
- Telegram:
  - búsqueda por texto
  - respuesta con poster + ficha resumida

## Fase 2
- Enriquecimiento TMDB
- Posters / backdrops
- filtros avanzados
- últimos añadidos
- favoritos / tags

## Fase 3
- índice incremental
- sincronización automática
- enlaces profundos desde Telegram a la web
